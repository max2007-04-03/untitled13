# Гра "Міста" (Cities Game)

## Опис
Гра в міста проти комп'ютера з графічним інтерфейсом Swing.

## Налаштування та запуск
1. **Файл міст**: Переконайтеся, що файл `cities.txt` знаходиться в ресурсах проєкту.
2. **Запуск**: Програма запускається через клас `ua.opnu.AppLauncher`.
3. **Версія Java**: Використовуйте Java 21 LTS.

## Правила гри
* Потрібно вводити реальні міста України
* Повторювати слова не можна
* Для завершення гри та перегляду рахунку введіть "здаюсь"
