# Гра "Міста" (Cities Game)

## Опис
Гра в міста проти комп'ютера з графічним інтерфейсом Swing.

## Налаштування та запуск
1. [cite_start]**Файл міст**: Переконайтеся, що файл `cities.txt` знаходиться в корені проєкту (папка `untitled13`).
2. [cite_start]**Запуск**: Програма запускається через клас `ua.opnu.AppLauncher`.
3. [cite_start]**Версія Java**: Використовуйте Java 21 LTS[cite: 6].

## Правила гри
* [cite_start]Потрібно вводити реальні міста України[cite: 25, 28].
* [cite_start]Повторювати слова не можна[cite: 30].
* [cite_start]Для завершення гри та перегляду рахунку введіть "здаюсь"[cite: 34, 35].
